# pdcurses-d
[D](https://dlang.org/) bindings for [PDCurses](https://wmcbrine.com/pdcurses/).

## Installation
Add `"pdcurses-d": "~>1.0.0"` to your dub dependencies.

## Usage
Simply use it as you would use PDCurses:

```d
import pdcurses;

// be happy, or not because it's curses
```
